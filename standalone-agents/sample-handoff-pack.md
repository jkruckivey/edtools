# Sample Handoff Pack

Use this to test the Builder Agent. Copy everything below and paste it as input.

---

```
HANDOFF PACK

Course format: Self-paced

Module:
  Number: 1
  Title: Introduction to Marketing Analytics

Outcomes practiced:
  - MLO 1: Explain the role of analytics in modern marketing decision-making
  - MLO 2: Identify common marketing analytics metrics and what they measure
  - MLO 3: Interpret basic campaign performance reports
  - MLO 4: Distinguish between vanity metrics and actionable metrics
  - MLO 5: Recommend simple optimization actions based on campaign data

Element sequence:
  1. Hook / Bridge-in (Text)
  2. Learning Outcomes Widget
  3. Teaching Block 1 (MLO 1)
  4. Practice Quiz 1 (MLO 1) — 1 question
  5. Teaching Block 2 (MLO 2)
  6. Tiles: Common Marketing Metrics (MLO 2)
  7. AI Chat Widget: Marketing Analytics Expert
  8. Teaching Block 3 (MLO 3)
  9. Interactive Widget: Campaign Report Explorer (MLO 3)
  10. Source Accordion: Industry Benchmarks
  11. Teaching Block 4 (MLO 4)
  12. Practice Quiz 2 (MLO 4) — 2 questions
  13. Teaching Block 5 (MLO 5)
  14. Practice Activity: Optimization Recommendations (MLO 5)
  15. Go Deeper Accordion: Marketing Analytics Resources
  16. Module Reflection
  17. Final Assessment Connection
  18. Transition Note

Constraints:
  - Time target: 45 minutes
  - Special requirements: None
  - Known widget availability: Campaign Report Explorer (TO BE BUILT)

Content notes:
  - Focus on digital marketing context (social, search, email campaigns)
  - Use realistic but fictional campaign data for examples
  - Target audience: Marketing professionals new to analytics
```

# Alternate Handoff Pack (Cohort Format)

Use this to test cohort terminology (WLO instead of MLO).

---

```
HANDOFF PACK

Course format: Cohort

Module:
  Number: 1 (Week 1, Module 2)
  Title: Understanding Your Customer Base

Outcomes practiced:
  - WLO 1.1: Identify key customer segmentation variables
  - WLO 1.2: Analyze customer data to create persona profiles
  - WLO 1.3: Apply segmentation insights to marketing decisions

Element sequence:
  1. Hook / Bridge-in (Text)
  2. Learning Outcomes Widget
  3. Teaching Block 1 (WLO 1.1)
  4. Video: Customer Segmentation Fundamentals (3 min)
  5. Tiles: Segmentation Variables (WLO 1.1)
  6. AI Chat Widget: Customer Insights Expert
  7. Teaching Block 2 (WLO 1.2)
  8. Interactive Widget: Persona Builder Tool (WLO 1.2)
  9. Source Accordion: Segmentation Research
  10. Practice Quiz (WLO 1.1, 1.2) — 3 questions
  11. Teaching Block 3 (WLO 1.3)
  12. Practice Activity: Segmentation Application (WLO 1.3)
  13. Go Deeper Accordion: Customer Analytics Resources
  14. Module Reflection
  15. Anchor Project Connection
  16. Transition Note

Constraints:
  - Time target: 30 minutes
  - Special requirements: Video requires production
  - Known widget availability: Persona Builder Tool (TO BE BUILT)

Content notes:
  - B2B and B2C examples
  - Connect to Week 1 Anchor Project milestone
  - Reference industry frameworks (RFM, behavioral segmentation)
```
