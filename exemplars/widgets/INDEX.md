# Widget Exemplar Index

Quick reference for selecting appropriate exemplars based on complexity and use case.

## Complexity Legend
- **Simple**: <300 lines, 0-3 inputs, minimal interactivity
- **Standard**: 300-700 lines, 4-8 inputs, moderate interactivity
- **Complex**: 700+ lines, 9+ inputs, multi-section/advanced features
